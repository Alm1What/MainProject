Функціональність API
1. Аутентифікація та авторизація
- Реєстрація користувача (email, пароль, ім'я) +
- Логін користувача (JWT-токен) +
- Захист маршрутів через Spring Security (доступ до ресурсів тільки після авторизації) +

2. Управління користувачами
- Отримання інформації про поточного користувача +
- Отримання списку всіх користувачів (для адміністратора) +
- Оновлення даних профілю +
- Видалення акаунта +
